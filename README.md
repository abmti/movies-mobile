
Projeto The Movies APP

- Start Metro Bundle:
npx react-native start --reset-cache

- IOS Pod Install:
npx pod-install ios

- Run IOS:
npx react-native run-ios --simulator="iPhone 11 Pro"

- Run Android:
npx react-native run-android

- Run Tests:
yarn test
